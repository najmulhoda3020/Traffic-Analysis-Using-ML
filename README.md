# Traffic-Analysis-Using-ML
Traffic accident analysis & volume analysis using machine learning to recommend the safest, along with the shortest path <br>
Project executed in 3 parts<br>
Part 1:- EDA on Traffic accident, K-mean for accident hotspot clustering, training 3 different ML models for severity prediction of accident.<br>
Part 2:- EDA on Traffic volume, training Random forest classifier for volume prediction.<br>
Part 3:- Risk engine and Route Recommendation system
